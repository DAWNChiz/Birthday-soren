## How to Put it on GitHub (Free Hosting!)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
- Click the **+** button → **New repository**
- Name it: `birthday-soren` (or anything you want)
- Make it **Public**
- Click **Create repository**

### Step 3 — Upload your files
- Click **uploading an existing file**
- Drag and drop the entire `birthday-soren` folder contents
- (Make sure the `photos/` folder and all photos are included!)
- Click **Commit changes**

### Step 4 — Enable GitHub Pages
- Go to your repo → **Settings** tab
- Click **Pages** in the left sidebar
- Under **Source**, select `main` branch and `/ (root)` folder
- Click **Save**

### Step 5 — Get your link!
After ~2 minutes, your site will be live at:
```
https://YOUR_USERNAME.github.io/birthday-soren
```

Send that link to Soren on April 16th! 🎂
